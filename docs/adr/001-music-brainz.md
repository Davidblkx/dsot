# How to query music brainz

## Context

As of today (05/2026) there are no library in rust that covers all API for music brainz. So I can eitheir create my own library and add what I need, or I could help evolve a current solution (`musicbrainz_rs` or `musicbrainz_rs_nova`)

## Decision

Since there's no library with all the features I need, it's easier to just create one and work in the same repo without requiring to wait for PR to be accepted.
