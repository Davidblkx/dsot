# Diagrams

Here you can find the Mermaid source files for all architecture and context diagrams used across the DSOT documentation:

- [context.mermaid](file:///projects/dsot/docs/diagrams/context.mermaid): High-level system context (L1 context).
- [containers.mermaid](file:///projects/dsot/docs/diagrams/containers.mermaid): Cargo Workspace container architecture (L2 containers).
- [data_location_resolution.mermaid](file:///projects/dsot/docs/diagrams/data_location_resolution.mermaid): Flowchart detailing how the configuration loaders resolve the storage data folder location.
- [state_initialization.mermaid](file:///projects/dsot/docs/diagrams/state_initialization.mermaid): Flowchart of the sequential state startup chain in `dsot_lib`.
- [platform_bootstrapping.mermaid](file:///projects/dsot/docs/diagrams/platform_bootstrapping.mermaid): Flowchart of the compilation-feature platform initialization hooks in `dsot_app`.
